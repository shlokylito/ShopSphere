# ShopSphere

A full-stack e-commerce platform built using React.js, Node.js, Express.js, MongoDB, Stripe, Redis, and Cloudinary.

## Features
- User authentication
- Product catalog
- Shopping cart
- Stripe payments
- Admin dashboard
- Coupon system
- Sales analytics

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Redis
- Stripe
- Cloudinary
